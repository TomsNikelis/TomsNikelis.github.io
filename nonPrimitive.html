String data tips
String data tips ir simbolu masīvs. String ir paradzēts, lai uzglabātu vairākus simbolus vienā mainīgajā. String maksimālais simbolu skaits ir 2^31-1.
Array data tips
Array, jeb latviski ir masīvs. Masīvs ir datu struktrūra, kas sastāv no viena data tipa elementiem. Maksimālais elementu skaits ir 10^7.
