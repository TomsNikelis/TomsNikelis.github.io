Datu tipi

Data tipi ir data tips, kas pasaka kompilatoram, kādu data tipu vēlas izmantot. Nosaucot data tipi ir iespēja ar to veikt matemātiskas darbības balstoties uz to. Ir divas data tipu grupas - Non-Primitive un Primitivi.

Non-Primitive Data tipi
Non-Primitive datu tipus Java sauc arī par atsauces tipiem, jo ​​tie attiecas uz objektiem. Ir diva veida Non-Primitive data tipi - Array un String.
Primitive data tipi
Primitīvs datu tips ir, kas iebūvēts javā, vai tāds, kuru varētu raksturot kā pamata struktūru sarežģītāku datu tipu veidošanai.
